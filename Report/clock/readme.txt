=========================================
     Mechanical Clock - Version 1.00

   製作者：4年 電子情報工学科 福澤大地
=========================================


◇ 概要 ◇
歯車が透けて見える振り子時計です。
リアルで高級感のある時計に仕上げました。


◇ ファイル構成 ◇
clock.c    … メインのソースファイル
clock.h    … メインのヘッダファイル
shape.c    … 図形描画のソースファイル
shape.h    … 図形描画のヘッダファイル
dial.png   … 文字盤の画像
icon.ico   … アイコンの画像
icon.rc    … アイコン用のリソース
Makefile   … make用のルールを記述したファイル
readme.txt … 本ファイル


◇ 動作確認環境 ◇
・Windows 10 Home 64bit
・Cygwin 3.0.7
・GCC 7.4.0
・GLUT 3.7


◇ ビルドと実行方法 ◇
GLUTとglpngが導入されているCygwin上で、次のコマンドを入力することでビルド、実行することができます。

$ make
$ ./j16426.exe
